# Plot the Results of the Non-linear Test

**\[deprecated\]** Soft-deprecated in favour of
[`autoplot.nonlintest()`](autoplot.nonlintest.md), which returns a
ggplot object you can extend with `+`. The base R-style plot below still
works.

## Usage

``` r
# S3 method for class 'nonlintest'
plot(x, plot = lifecycle::deprecated(), ...)
```

## Arguments

- x:

  a `nonlintest` object produced by [`nonlintest()`](nonlintest.md).

- plot:

  **\[deprecated\]** display plot (TRUE) or return plot (FALSE).

- ...:

  additional arguments to
  [`plot()`](https://rdrr.io/r/graphics/plot.default.html)

## Value

ggplot Contour plot of the region of the third-order moment outside the
test limits generated by [`nonlintest()`](nonlintest.md).

## See also

[`autoplot.nonlintest()`](autoplot.nonlintest.md),
[`nonlintest()`](nonlintest.md)

# \donttest{
